Web development

From coding HTML and CSS to design aesthetically pleasing user interfaces to implementing complex functionality through dynamic programming languages ​​like JavaScript, web development is a complex dance between aesthetics and functionality.

Angular

Angular offers a solid framework for building dynamic web applications, with a rich set of features out of the box. For this I created a web application with Angular called Blog and another called Crud_Hero

NodeJs

One of the main advantages of Node.js is its ability to unify client-side and server-side development using the same language, JavaScript. This allows for a smooth transition from browser features to server features, simplifying the development process.

Laravel

One of Laravel's most notable features is its intuitive routing system, which simplifies handling URLs and HTTP requests. In addition, it offers a database abstraction layer, called Eloquent ORM, which facilitates the manipulation of data in the database. With Laravel I developed 2 web applications Car Rental Management and Crud Students
